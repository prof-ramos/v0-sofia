## Purpose

Assets estáticos servidos diretamente: favicons, logos e placeholders.

## Key Files

| File | Description |
|------|-------------|
| `logo.svg` | Logo institucional da ASOF (usado no header) |
| `favicon.svg` | Favicon principal (referenciado no layout) |

| `favicon-dark.png` | Favicon para tema escuro |

## For AI Agents

### Working In This Directory

- Não modificar assets sem aprovação do design
- `logo.svg` é referenciado em `components/sofia/header.tsx`
- Favicons configurados em `app/layout.tsx` (metadata.icons)
- Placeholders (`placeholder-*`) são artefatos do v0 scaffold, podem ser removidos
